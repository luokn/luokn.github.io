经典机器学习算法的极简实现

2021年10月10日 · 阅读需 6 分钟
Kun Luo
Software Engineer

大三时曾花两周时间集中学习经典机器学习算法：白天对照李航老师的《统计学习方法》推导公式，晚上把公式一行行写成代码。 在参考 GitHub 上现有实现时，我发现很多代码为了追求“工程化”而变得冗长，反而掩盖了算法本身的核心思想。 于是我把自己的实现整理出来，尽量保持极简、可读、可运行，希望能帮助大家快速理解算法原理。

项目地址：

写在前面​

这套实现有几个特点：

  • 零依赖：除 NumPy、Matplotlib 和 SciPy 外，不调用任何机器学习库；
  • 代码极简：每个算法都压缩到最核心的几十行，方便对照公式阅读；
  • 附带可视化：大部分算法都配了生成数据和绘制结果的脚本，跑起来就能看到效果；
  • 对应教材：实现思路主要参考《统计学习方法》，适合边看书边敲代码。

当然，这种写法重在教学，不追求工业级效率。理解了原理之后，实际项目中还是建议使用 scikit-learn、XGBoost、PyTorch 等成熟工具。

核心代码片段​

下面挑选三个最具代表性的算法，展示它们的“骨架”实现。

PCA：主成分分析​

PCA 的核心只有三步：去中心化 → 协方差矩阵特征值分解 → 取前 kkk 大特征值对应的特征向量。

class PCA:
    def __init__(self, k: int):
        self.k = k

    def __call__(self, X: np.ndarray):
        X_norm = X - X.mean(axis=0)               # 1. 去中心化
        L, V = np.linalg.eig(X_norm.T @ X_norm)   # 2. 协方差矩阵特征值分解
        topk = np.argsort(L)[::-1][: self.k]      # 3. 前 k 大特征值索引
        return X_norm @ V[:, topk]                # 4. 投影到低维空间

Perceptron：感知机​

感知机是神经网络的“鼻祖”。它逐个样本检查是否分类正确，若错误则沿负梯度方向更新权重。

class Perceptron:
    def __init__(self, input_dim: int, lr=5e-4):
        self.weights = np.random.randn(input_dim + 1)  # 包含偏置
        self.lr = lr

    def fit(self, X: np.ndarray, y: np.ndarray):
        for x, y in zip(pad(X), y):
            if y * (x @ self.weights) <= 0:        # 分类错误
                neg_grad = x * y                    # 计算负梯度
                self.weights += self.lr * neg_grad  # 沿负梯度更新

    def __call__(self, X: np.ndarray) -> np.ndarray:
        y_pred = pad(X) @ self.weights
        return np.where(y_pred > 0, 1, -1)

Logistic Regression：逻辑斯蒂回归​

逻辑回归用 sigmoid 把线性输出映射为概率，再用梯度下降最小化交叉熵损失。

class LogisticRegression:
    def __init__(self, input_dim: int, lr=5e-4):
        self.weights = np.random.randn(input_dim + 1)
        self.lr = lr

    def fit(self, X: np.ndarray, y: np.ndarray):
        X_pad = pad(X)
        pred = sigmoid(X_pad @ self.weights)         # 预测概率
        grad = X_pad.T @ (pred - y) / len(pred)      # 交叉熵梯度
        self.weights -= self.lr * grad               # 参数更新

    def __call__(self, X: np.ndarray) -> np.ndarray:
        y_pred = sigmoid(pad(X) @ self.weights)
        return np.where(y_pred > 0.5, 1, 0)

一点学习心得​

初学机器学习时，很容易被各种框架和调参技巧绕进去。我的体会是：先回到公式，再用最短的代码把它实现出来，远比直接调包来得扎实。

例如，当你亲手写过 SVM 的 SMO、写过 GMM 的 EM、写过决策树的信息增益，你就会真正理解：

  • 为什么 SVM 只依赖支持向量；
  • 为什么 EM 算法要交替进行 E 步和 M 步；
  • 为什么决策树容易过拟合，需要剪枝或限制深度。

这套代码就是我当时“啃”《统计学习方法》的副产品。希望它也能成为你入门机器学习路上的一块垫脚石。
