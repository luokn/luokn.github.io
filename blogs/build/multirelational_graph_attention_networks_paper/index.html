MS-GAT：用多关系图注意力网络学习多层面交通耦合

1. 背景：交通预测难在哪里？​

实时准确的交通预测是智能交通系统（ITS）的核心任务之一，对动态路径规划、信号控制和出行引导都有重要价值。然而，真实道路网络中的交通状况受到多种复杂因素的共同影响：

  • 时间维度：同一检测点在不同时刻的流量、速度、占有率存在周期性和趋势性依赖；
  • 空间维度：相邻路口、上下游路段的交通状态会相互传播；
  • 多通道耦合：同一个节点往往部署了多种传感器（流量、速度、密度等），这些信号在现实中并非独立，而是彼此耦合、相互影响。

传统方法如 ARIMA、SVR、VAR 等只能建模简单的时序依赖；CNN 和 RNN 分别擅长空间或时间特征，但难以同时处理两者的交互。近年来，图神经网络（GNN）将道路网络建模为图，在交通预测上取得了显著进展，例如 STGCN、DCRNN、Graph WaveNet、AGCRN 等方法都取得了很好的效果。

但这些方法大多只关注时空关系，而忽略了交通信号在不同通道之间的耦合关系。我们提出将交通系统看作一个耦合交通网络（Coupled Traffic Network, CTN），显式建模多层面交通信号的耦合关系。

2. 问题定义：耦合交通网络​

我们将道路网络建模为一个有向图 G=(V,E,A)G=(V, E, A)G=(V,E,A)，其中 VVV 是交通节点（如检测器位置），EEE 是边，A∈RN×NA \in \mathbb{R}^{N \times N}A∈RN×N 是邻接矩阵。

每个节点在同一时刻会观测到多种交通信号（流量、速度、占有率等），我们把每一种信号看作一个通道（channel）。于是，交通条件 XXX 可以从三个维度理解：

  • 通道维度（Channel）：同一节点不同信号之间的相互影响，构成通道关系；
  • 时间维度（Temporal）：同一节点同一信号在不同时刻的演变，构成时间相关性；
  • 空间维度（Spatial）：同一时刻不同节点之间的交通状态传播，构成空间相关性。

3.1 多组件时间门控融合​

交通预测本质上是一个序列到序列（seq2seq）的学习任务。我们发现：

  1. 一次性输入过长的历史序列不一定能提升预测精度，反而会增加模型开销；
  2. 不同历史时段对未来同一时刻的影响强度并不相同。

因此，MS-GAT 采用多组件（multi-component）结构。每个预测组件（Traffic Prediction Component，TPC）只处理一段较短的历史时间序列，多个 TPC 分别对应不同偏移量的历史时段。通过一个时间嵌入门控（Time Embedding Gate），模型自适应地融合各 TPC 的输出：

y^=∑i=1Mgi⊙fi(xi)\hat{y} = \sum_{i=1}^{M} g_i \odot f_i(x_i)y^​=i=1∑M​gi​⊙fi​(xi​)

其中 gi=TE(Di,Hi)g_i = TE(D_i, H_i)gi​=TE(Di​,Hi​) 由星期几 DiD_iDi​ 和时刻 HiH_iHi​ 编码得到，⊙\odot⊙ 表示逐元素乘积。这种设计既保留了各时段信息的差异性，又通过门控机制实现了自适应融合。

3.2 多维自注意力机制​

为了让模型方便地处理三维数据（通道、时间、空间），我们提出了一种多维自注意力（Multi-dimensional Self-Attention）机制。相比传统自注意力只作用于单一维度，我们的方法可以复用到任意维度。

以三维数据 {hx,y,z}\{h_{x,y,z}\}{hx,y,z​} 为例，若希望捕捉 xxx 维度的相关性，可将查询和键设置为 hx,:,:h_{x,:,:}hx,:,:​，并通过线性变换得到 Q□Q_{\square}Q□​ 和 K□K_{\square}K□​。注意力图计算为：

A□=softmax(Q□W□K□⊤)A_{\square} = \text{softmax}(Q_{\square} W_{\square} K_{\square}^{\top})A□​=softmax(Q□​W□​K□⊤​)

为了避免节点数或时间步过大导致参数爆炸，我们进一步将 W□W_{\square}W□​ 分解为两个低秩矩阵的乘积 W□=E□E□⊤W_{\square} = E_{\square} E_{\square}^{\top}W□​=E□​E□⊤​，从而显著降低参数量。

3.3 MEAM：多关系嵌入并列模块​

MS-GAT 的核心是 Multi-relational Embedding Abreast Module（MEAM）。每个 MEAM 内部有三个并列分支，分别学习：

  • 空间关系嵌入：通过 GAttention 计算节点间注意力图，再用 GCN 聚合邻居信息；
  • 时间关系嵌入：通过 TAttention 捕捉时序依赖，再用 TCN（时序卷积网络）建模因果时序关系；
  • 通道关系嵌入：通过 CAttention 学习不同交通信号通道之间的耦合关系。

三个分支的输出在层间拼接后送入下一层 MEAM，下一层的自注意力机制会隐式地学习各关系对预测结果的相对重要性。通过堆叠多个 MEAM，模型能够同步、自适应地区分时空关系和通道关系的贡献。

4. 实验结果​

我们在五个真实交通数据集上评估了 MS-GAT，包括 PEMSD3、PEMSD4、PEMSD7、PEMSD8 和 PEMS-BAY。这些数据集均来自加州 Caltrans PeMS 系统，时间粒度为 5 分钟。

与 DCRNN、STGCN、ASTGCN、Graph WaveNet、STSGCN、AGCRN 等六种主流方法相比，MS-GAT 在绝大多数指标上取得了最优或接近最优的结果。例如，在 PEMSD3、PEMSD4、PEMSD7、PEMSD8 上，MS-GAT 的 MAE 分别比当时最好的方法提升了 7.0%、1.4%、7.8% 和 12.6%。

图 5：MS-GAT 在不同设置下的 MAE 敏感性分析

消融实验进一步验证了三个关键设计的有效性：

  1. 多组件结构：适当增加 TPC 数量能提升预测精度，但并非越多越好，需根据具体路网复杂度调整；
  2. 时间嵌入（TE）：引入时间嵌入后模型性能 consistently 提升，说明不同历史时段对未来预测的贡献不同；
  3. 通道关系（CAttention）：去掉通道注意力后性能明显下降，证明了通道关系在交通预测中的重要性。

案例研究​

我们选取了 PEMSD4 中的两个节点（Node 0 和 Node 200）进行可视化。结果显示，MS-GAT 的预测曲线与真实交通流量趋势更加吻合，尤其是在 Node 200 这种波动相对规律的节点上优势明显。Node 0 由于存在频繁的随机交通事件（如短时拥堵-疏通交替），预测难度更大，但 MS-GAT 仍表现出更好的适应性。

5. 总结​

MS-GAT 的核心贡献在于：

  1. 首次在交通预测中显式建模通道关系，将交通系统视为通道、时间、空间三维耦合的复杂网络；
  2. 提出了多维自注意力机制，使自注意力能够方便地扩展到多维数据；
  3. 设计了多关系嵌入并列模块（MEAM），同步学习并自适应融合三种关系。

在五个真实数据集上的实验表明，MS-GAT 能够有效捕捉交通信号之间的多层面耦合关系，在短时和长时预测任务上都优于多种主流基线方法。
